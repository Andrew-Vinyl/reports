What is Engagement Depth?

Engagement Depth Score is a strictly normalized performance index (0-100 scale) that measures genuine content consumption. It rigorously isolates high-quality engagement by factoring the true Average Engagement Time per Active User to highlight exactly which topics are holding executive attention and building trust.
